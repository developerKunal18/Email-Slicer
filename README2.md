# Email Slicer

## Overview
The **Email Slicer** is a simple Python program that breaks an email address into its main components.

It extracts the username, domain name, and extension, making it a great beginner project for practicing string manipulation.

---

## Features
- Extracts username from email
- Extracts domain name
- Extracts domain extension
- Simple input validation
- Clean command-line output

---

## Technologies Used
- Python
- String methods
- Conditional statements
- User input handling

---

## How It Works
1. User enters an email address  
2. Program checks basic validity  
3. Email is split using `@` and `.`  
4. Components are displayed clearly
